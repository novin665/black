 نصب ربات
```sh
cd $HOME
git clone https://github.com/novin665/black.git
cd black 
chmod +x install
chmod +x start
./install
./start
# سپس شماره و کد را ارسال کنید :)
# اجرای اتولانچ:
cd black 
chmod 777 screen
chmod +x install
./install
screen ./screen
```
